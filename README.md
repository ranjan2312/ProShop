# Redshop
MernStack Ecommerce website


1.Install Dependencies (frontend & backend)
npm install
cd frontend
npm install

2. Run
# Run frontend (:3000) & backend (:5000)
npm run dev

# Run backend only
npm run server


#.# Import data
npm run data:import

# Destroy data
npm run data:destroy
